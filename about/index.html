About

Hi there! =)

My name is Alen (aka Devgaze), I am a full stack web developer by profession, sailor in my heart, and enthusiast for a lot more!

This is my personal web space, where I publish technology related thinkings and findings. It is planned to be one stop guide for myself, regarding interesting concepts and everyday solutions I encounter throughout my professional career.

And if you, my dear reader, find it useful too... it becomes so much more valuable to me.

A brief overview of topics I will cover through my blog:

  • Front end development
    • HTML, CSS, Javascript
    • AngularJS, React, etc.
  • Back end development
    • Python
    • Django and related packages
    • Virtualenv
  • Methodologies
    • TDD (Test Driven Development)
    • CI (Continous integration)
    • Agile project management (SCRUM)
  • Sailing

And finally, DON'T be a stranger. Feel free to contact me about anything, especially if you stumble on a bad user experience inside of these pages.
